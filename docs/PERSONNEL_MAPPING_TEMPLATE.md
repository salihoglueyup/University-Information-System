## Doldurma Sonrası Prosedür

1. Tabloyu tamamla (tüm sütunlara veri gir)
2. Aşağıdaki Python betiği çalıştır (FEATURES.md'yi otomatik güncelle):

```python
# update_features_ownership.py
import csv
import re

# personnel_mapping.csv dosyasından oku
mapping = {}
with open('docs/personnel_mapping.csv') as f:
    reader = csv.DictReader(f)
    for row in reader:
        team = row['Takım'].strip()
        owner = row['Atanan Kişi'].strip()
        if team and owner:
            mapping[team] = owner

# FEATURES.md'yi oku ve güncelle
with open('docs/FEATURES.md') as f:
    content = f.read()

# Her takım için replacements yap
for team, owner in mapping.items():
    old_pattern = f"Product Owner - [^|]*\s*\|\s*{re.escape(team)}"
    new_text = f"{owner} | {team}"
    content = re.sub(old_pattern, new_text, content)

# Güncellenen dosyayı kaydet
with open('docs/FEATURES.md', 'w') as f:
    f.write(content)

print("✅ FEATURES.md güncellendi!")
```

3. Git commit yap:
```bash
git add docs/FEATURES.md docs/PERSONNEL_MAPPING.md
git commit -m "docs: assign personnel ownership to feature groups"
```
