# Student Guide

A comprehensive guide for students using the UBIS Student Information System.

## Getting Started

### Logging In

1. Navigate to the UBIS login page
2. Enter your **student number** (e.g., `B211200051`) as username
3. Enter your password
4. If 2FA is enabled, enter the 6-digit code from your authenticator app

> **First-time users:** Your initial password is set by the university administration. Change it immediately after your first login via **Settings → Account**.

### Dashboard Overview

After login, you'll see the main dashboard with:

- **Quick Stats** — GPA, enrolled courses, upcoming assignments
- **Schedule Widget** — Today's classes at a glance
- **Announcements** — Latest university and department announcements
- **Notification Bell** — Real-time notifications for grades, assignments, and announcements

## Common Workflows

### Viewing Your Grades

1. Navigate to **Akademik → Ders Notlarım**
2. View your current semester grades (Vize, Final, Bütünleme)
3. Letter grades and GPA are calculated automatically
4. For your full academic history, use **Transkript**

### Submitting an Assignment

1. Navigate to **Akademik → Ödevlerim**
2. Find the assignment you need to submit
3. Click the assignment to open submission form
4. Upload your file (max 10MB)
5. Click "Gönder" to submit

> **Note:** Assignments past their due date will be marked as "Gecikti" (Overdue) automatically at midnight.

### Course Registration

1. Navigate to **İdari → Ders Kayıt İşlemleri**
2. Review available courses for the upcoming semester
3. Check prerequisites via **Ders Ön Koşulları**
4. Select courses and confirm registration
5. Check **Akademik Takvim** for registration deadlines

### Checking Payments

1. Navigate to **İdari → Finans ve Cüzdan**
2. View your current balance and tuition status
3. See transaction history
4. Use the payment gateway for tuition payment

### Borrowing a Book

1. Navigate to **Kampüs → Kütüphane**
2. Browse or search the book catalog
3. Click "Borrow" on an available book
4. Return before the due date (you'll receive email reminders 3 days before)

### Setting Up 2FA

1. Navigate to **Genel → Hesap Ayarları**
2. Find the "Two-Factor Authentication" section
3. Click "Enable 2FA"
4. Scan the QR code with Google Authenticator or similar app
5. Enter the 6-digit verification code
6. 2FA is now active for your account

## Tips

- 💡 **Dark Mode**: Toggle via Settings or the theme switcher
- 🔔 **Notifications**: Real-time alerts appear as toast popups when grades, assignments, or announcements are posted
- 📱 **PWA**: Install UBIS as a Progressive Web App on your phone for quick access
- 🤖 **AI Assistant**: Use the floating AI widget (bottom-right) for quick questions about the system
